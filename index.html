Challenge HTML Dia II

Tiempo destinado: 3hrs


Terminé el nav, agregué una función con jquery para que se oculte en la vista mobile.Haciendo un click en el icono de menu se despliega nuevamente. Enlacé el archivo script.js en el index. Y modifiqué con Flexbox algunos aspectos de la página. Aprendí a usar media queries para adaptar el nav en tamaños pequeños de pantalla, integré GoogleFonts y FontAwesome y coloqué el logo con link en formato png.

Próximas Actualizaciones

  • Agregar un footer
  • Agregar un formulario
  • Aprender sobre CSS Grid
  • Registrar distintas propiedades de estilos en CSS
  • Incoporar fotos en la página
  • Tenerme mucha paciencia y registrar los avances

Challenge HTML Dia III

Tiempo destinado: 1hrs

Incluso agregué una animación de una libreria con el fin de entender un poco como se codifica. Vi algunos repsositorios en GitHub sobre el tema y leí mucho código.Agregué el footer.

  • Tenerme mucha paciencia y registrar los avances
